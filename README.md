# BoardBook

Smart Classroom Solution - Interactive Board Launcher

BoardBook은 스마트 교실 환경을 위한 올인원 런처 솔루션입니다. 화이트보드, PDF 뷰어, 미디어 플레이어 등 수업에 필요한 다양한 도구를 하나의 인터페이스에서 통합 관리할 수 있습니다.

## 🚀 주요 기능

- **All-in-One Dashboard**: 수업 도구 및 콘텐츠에 즉시 접근 가능한 통합 인터페이스
- **Smart Tools**: 화이트보드, 이미지/비디오 플레이어, PDF 뷰어 내장
- **Classroom Optimization**: 교실 환경에 최적화된 사용자 경험 및 성능
